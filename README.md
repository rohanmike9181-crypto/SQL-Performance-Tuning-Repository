# SQL Server Performance Tuning

Comprehensive SQL Server performance tuning and troubleshooting repository containing production-ready T-SQL scripts for identifying high CPU queries, blocking sessions, missing indexes, deadlocks, TempDB usage, long running queries, wait statistics, and execution plan analysis.

---

## Features

- High CPU query identification
- Blocking session analysis
- Missing index detection
- Deadlock troubleshooting
- TempDB monitoring
- Long running query analysis
- Query performance optimization
- SQL Server troubleshooting scripts
- Production DBA monitoring

## Example Scripts Included

### Top CPU Consuming Queries
Identifies queries consuming excessive CPU resources.

### Blocking Sessions
Detects blocking chains and blocked sessions.

### Missing Indexes
Displays recommended indexes for performance improvement.

### TempDB Usage
Monitors TempDB space consumption.

### Long Running Queries
Identifies currently executing long-duration queries.

### Deadlock Information
Retrieves deadlock graph information from system health session.
